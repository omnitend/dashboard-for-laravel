# DXDashboardNavbar

A responsive top navigation bar component for Laravel dashboards with user menu, search functionality, and sidebar toggle.

## Live Examples

<ComponentExample
  client:load
  code={`<script setup>
import { ref } from 'vue';
import {
  DXDashboardNavbar,
  DDropdownItem,
  DDropdownDivider
} from '@omnitend/dashboard-for-laravel';

const currentUser = ref({
  name: 'Sarah Johnson',
  email: 'sarah@example.com',
});

const pageTitle = ref('Dashboard Overview');

const handleToggle = () => {
  console.log('Sidebar toggle clicked');
};

const handleLogout = () => {
  console.log('Logout clicked');
};
</script>

<template>
  <DXDashboardNavbar
    :user="currentUser"
    :page-title="pageTitle"
    @toggle-sidebar="handleToggle"
  >
    <template #search>
      <input
        type="text"
        class="form-control form-control-sm"
        placeholder="Search..."
      />
    </template>

    <template #user-menu-items="{ user }">
      <DDropdownItem href="/profile">
        Profile
      </DDropdownItem>
      <DDropdownItem href="/settings">
        Settings
      </DDropdownItem>
      <DDropdownDivider />
      <DDropdownItem @click="handleLogout">
        Log out
      </DDropdownItem>
    </template>
  </DXDashboardNavbar>
</template>`}
  example="DXDashboardNavbarExample"
/>

## Props

<PropsTable client:load props={[
  { name: 'user', type: '{ name: string; email: string; [key: string]: any } | null', required: 'No', default: 'null', description: 'Current user object with name and email' },
  { name: 'pageTitle', type: 'string', required: 'No', default: 'Dashboard', description: 'Page title displayed in the navbar' },
  { name: 'searchAlign', type: "'start' | 'center'", required: 'No', default: "'start'", description: 'Horizontal alignment of the search slot content within its region: flush left (start) or centred (center). The region is inline in the bar from md up, its own full-width row below' },
  { name: 'actionsOnMobile', type: "'wrap' | 'hide'", required: 'No', default: "'wrap'", description: 'What the actions slot does below md: wrap moves it to its own full-width row (the bar grows), hide removes it entirely (for apps that relocate page actions into the page on phones)' },
]} />

**Height note:** on desktop the bar is exactly 64px (aligned with the sidebar
header) as long as single-row content stays within its 48px budget (64px minus
the bar's padding) — default-size buttons and inputs fit. Taller content (e.g.
`btn-lg`, or a tall custom `user-menu` slot) makes the bar grow past 64px, so
it no longer lines up with the sidebar header.

## Extended Component

This is a custom component that extends beyond simple Bootstrap Vue Next wrappers,
providing additional functionality specific to Laravel dashboards.

## Features

- **Responsive Design**: Adapts to different screen sizes with mobile-friendly layout
- **User Avatar**: Automatically generates avatar with user initials
- **Flexible Search**: Customisable search slot for your search implementation
- **Sticky Positioning**: Stays at the top of the page when scrolling
- **Customisable Slots**: Multiple slots for complete customisation of layout and functionality
