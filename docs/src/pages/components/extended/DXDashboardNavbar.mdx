# DXDashboardNavbar

A responsive top navigation bar component for Laravel dashboards with user menu, search functionality, and sidebar toggle.

## Live Examples

<ComponentExample
  client:only="vue"
  code={`<script setup>
import { ref } from 'vue';
import {
  DXDashboardNavbar,
  DDropdownItem,
  DDropdownDivider
} from '@omni-tend/dashboard-for-laravel';

const currentUser = ref({
  name: 'Sarah Johnson',
  email: 'sarah@example.com',
});

const pageTitle = ref('Dashboard Overview');

const handleToggle = () => {
  console.log('Sidebar toggle clicked');
};

const handleLogout = () => {
  console.log('Logout clicked');
};
</script>

<template>
  <DXDashboardNavbar
    :user="currentUser"
    :page-title="pageTitle"
    @toggle-sidebar="handleToggle"
  >
    <template #search>
      <input
        type="text"
        class="form-control form-control-sm"
        placeholder="Search..."
      />
    </template>

    <template #user-menu-items="{ user }">
      <DDropdownItem href="/profile">
        Profile
      </DDropdownItem>
      <DDropdownItem href="/settings">
        Settings
      </DDropdownItem>
      <DDropdownDivider />
      <DDropdownItem @click="handleLogout">
        Log out
      </DDropdownItem>
    </template>
  </DXDashboardNavbar>
</template>`}
>
  <DXDashboardNavbarExample client:load />
</ComponentExample>

## Extended Component

This is a custom component that extends beyond simple Bootstrap Vue Next wrappers,
providing additional functionality specific to Laravel dashboards.

## Features

- **Responsive Design**: Adapts to different screen sizes with mobile-friendly layout
- **User Avatar**: Automatically generates avatar with user initials
- **Flexible Search**: Customisable search slot for your search implementation
- **Sticky Positioning**: Stays at the top of the page when scrolling
- **Customisable Slots**: Multiple slots for complete customisation of layout and functionality
